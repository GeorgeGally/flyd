---
description: Synthesize captures about a topic into a structured learning document. Use after completing work to document what was learned.
argument-hint: "<topic>"
---
Run `npx tsx src/index.ts compound $ARGUMENTS` to search memory for relevant captures, synthesize them into a learning document (Context → What We Learned → Key Decisions → Patterns → Open Questions), and save as a compound capture.

Use this whenever you solve a non-trivial problem or complete a feature, to preserve learnings for future sessions.
